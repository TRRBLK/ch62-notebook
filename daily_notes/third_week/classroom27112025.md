#IA
Simula inteligencia humana

Machine Learning subconjunto clave de la IA aprende de los datos con 
el tiempo y mejoran su redinmiento sin necesidad de programar 
explicitamente.

Deep learning es una forma avanzada de aprendizaje automatico que 
utiliza redes neuronales con muchas capas para procesar grandes 
cantidades de datos, lo que permite tareas como el reconocimiento de 
imagenes y la traducción de idiomas.

GenIA es una tecnologia que crea contenido nuevo como texto, 
imagenes, audio, video y codigo.

Datos -> Modelo -> Aprendizaje -> Salida (Predicçión/Decisión)

#SUPERVISADO
El modelo aprende a partir de datos etiquetados, cada entrada tiene 
una salida conocida.

#NO SUPERVISADO
El modelo aprende patrones o estructuras ocultas en los datos sin 
etiquetas. Descubre nuevas agrupaciones o reducir la dimension de 
datos.

#Refuerzo
El modelo aprende interactuando con un entorno. Toma acciones, 
recibe retroalimentación en forma de recompensas o penalizaciones, 
y ajusta su estrategia para maximizar los resultados positivos.

#DEEP LEARNING
Redes Neuronales Convolucionales (CNNs). Diseñada para reconocer 
patrones visuales, excelentes para analizar fotografias.

Redes Neuronales Recurrentes (RNNs). Procesan datos secuenciales 
recordando entradas anteriores para entender el contexto a lo 
largo del tiempo.

Transformadores. Modelos avanzados para el procesamiento de 
lenguaje, capaces de entender oraciones completas a la vez.

Los LLM son un tipo de modelo transformador especializado en 
entender y generar texto similar al humano.
