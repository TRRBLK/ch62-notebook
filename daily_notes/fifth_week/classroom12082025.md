# EMPLEABILIDAD
## Diseño de CV
Carta de presentación para destacar habilidades blandas y técnicas. Éste debe 
estar bien diseñado para cubrir la mayoría de los requisitos.

## Consideraciones
- Enfocar el CV al perfil deseado. Cumplimentar con los ATS. 
- Solo una hoja
- Ortografía

El documento es dinamico.

## Elementos
### Encabezado
Nombre completo, perfil profesional, Full-Stack Developer Jr | Desarrollador 
Full-Stack Jr. Acoplarse a las métricas de la vacante para ser evaluados por 
rubricas de ATS.

### Datos de contacto
Se debe cubrir los siguientes datos:
- Numero de contacto con código de país: +52
- Correo profesional
- Hipervinculo de: LinkedIn, GitHub y portafolio de proyectos
- Ubicación
- Adicional, colocar si estas disponible para reubicación

### Perfil profesional
Soy un desarrollador Java Full Stack ¿en donde estoy? ¿de donde vengo? ¿para 
donde voy?
- Descripción del perfil
- Habilidades
- Que se desea obtener con el empleo

### Habilidades
Adicionales de cursos, proyectos, y las que se obtienen a lo largo del 
bootcamp

### Proyectos academicos
Proyecto integrador y proyectos adicionales

### Experiencia profesional
Verbo de acción + ¿Qué hice? + ¿Para qué lo hice? ¿Con qué lo hice? + 
Logro cuantificable (agregar e-commerce)

Experiencia previa para dar valor agregado cronologicamente.

Colocar el bootcamp y formación previa, debe ir en orden cronologico.

Cursos: respetar el orden orden cronologico, mientras más cursos y 
certificaciones, mayor peso tendrá tu CV.
