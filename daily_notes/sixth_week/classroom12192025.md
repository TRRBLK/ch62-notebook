# Entendimiento y explicacion de conceptos basicos
Trasladar a lenguaje coloquial, mostrar capacidad de resolución, utilizar 
practicas de codificación: legibilidad, eficiencia, funcionalidad.

Algunas de las pruebas tienden a ser más desafiantes a próposito, 
para observar la capacidad de resolucion de problemas, de cómo se 
enfrentan a algo que puede que no sepan hacer en el momento.

# Desafio para desarrollar en casa
Se asigna una tarea para desarrollar con especificaciones y requerimientos. Se 
establece un tiempo de entrega.

Debe realizarse preguntas adicionales sobre la programación y funcionamiento para 
descartar plagio o alguna otra situación de ese tipo.
