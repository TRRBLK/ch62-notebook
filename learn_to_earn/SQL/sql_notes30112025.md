##BASE DE DATOS
Estructuras en las que se almacenan conjuntos de datos para su posterior 
procesamiento: consulta, análisis, relacionamiento o transmisión.

Una analogia a las BBDD son las bibiliotecas, donde el contenido se puede 
consultar por etiquetas.

Ejemplos son Guias telefonicas, redes sociales y transacciones bancarias.

##TIPO DE BASE DATOS
#SEGUN SU VARIEDAD
- Estáticas: solo se permite la lectura. Almacenan datos historicos para 
estudiar su comportamiento a lo largo del tiempo y realizar previsiones a 
futuro.

- Dinámicas: actualizan, reorganizan, añaden y eliminan información a lo 
largo del tiempo.

#SEGUN SU CONTENIDO
- Bibliograficas: recopilaciones cientifico-tecnico que reunen datos sobre 
un área de conocimiento.

- Directorios: listado de datos personales en formato electronico.

- De texto completo: recogen articulos completos, y cuya busqueda puede 
realizarse por palabras clave, fechas, etc.

- Especializadas: cubre necesidades puntuales de un publico objetivo que 
consume dicha informacion. 

#MODELOS DE BASE DE DATOS
Modelo relacional: datos relacionales, consiste en un esquema de tablas con 
filas y columnas (SQL)

Modelo jerarquico: parecida a un árbol, padre e hijo, compuesta por una 
colección de registros.

Modelo multidimensional.

Deductivo. Reglas y hechos almacenados.

Modelo OOP.

Modelo transaccionales. 

Data Base Management Systems.

#BBDD RELACIONAL
Permite interactuar con los datos mediante tablas para generar información 
adicional o volver a montar los datos de distintas maneras.
